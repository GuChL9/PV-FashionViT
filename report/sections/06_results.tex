\section{实验结果与量化分析}

本节使用六组模型在五个随机种子下的正式 run，不包含 smoke test。表\ref{tab:main-results}同时列出平移、旋转和复合扰动结果；每个模型的 Mean 行给出五次运行均值，$\pm$ Std 行给出跨种子标准差。由于不同指标回答的问题不同，后续分析不以单列排序代替整体判断。

\subsection{主结果：中心高分不等于位置鲁棒}

\input{tables/main_results}

中心训练的三种基线在 Center 上均达到 $86\%$ 以上，其中 CNN 最高，均值为 $90.93\%$。一旦测试分布改为 Large Shift，MLP、CNN 和 ViT-AbsPos 分别下降至 $14.22\%$、$13.80\%$ 和 $20.03\%$；对应 Robust Drop 为 73.41、77.13 和 66.35 个百分点。CNN 在中心位置表现最好，却拥有六组中最大的 Robust Drop。这说明局部权值共享可以支持常规分类和小范围模式提取，但不足以抵消固定中心训练、后端固定空间布局以及极端边缘裁剪的共同影响。

ViT-AbsPos 的中心准确率低于 CNN 4.55 个百分点，但 Large Shift 和 Grid Accuracy 分别高出 6.23 和 7.60 个百分点。全局 token 交互可能扩大了可利用的空间范围，不过 $66.35$ 个百分点的 Robust Drop 仍然很大。换言之，普通 ViT 并没有因为使用自注意力就自然获得位置不变性；中心训练分布和绝对位置编码依然留下了明显偏置。

增强模型呈现出不同的取舍。ViT-Aug 的 Center Accuracy 均值为 $62.81\%$，明显低于三个中心基线，但 Large Shift、Rotation 和 Shift+Rotation Accuracy 都约为 $65\%$。ViT-MeanPool 的对应指标为 $63.51\%$、$66.29\%$、$66.20\%$ 和 $66.83\%$，整体略高于 ViT-Aug；不过二者在 Grid Accuracy 上只相差 1.40 个百分点，且五个 seed 中有一次差值反向。因此，Mean Pooling 在这里更适合被看作轻微正向趋势，而不是主要改进来源。

HybridConv-ViT 在稳定性和绝对准确率之间取得更好的平衡：Center Accuracy 为 $81.42\%$，Large Shift 为 $81.33\%$，Rotation 为 $81.86\%$，Shift+Rotation 为 $80.90\%$，Grid Accuracy 为 $81.05\%$。它没有超过 CNN 的中心成绩，但在位置、角度和复合扰动协议下都保持在约 $81\%$--$82\%$，更符合本项目对“位置可变分类器”的定义。

\subsection{从增强到 Hybrid 的逐步变化}

\input{tables/ablation_results}

表\ref{tab:ablation-results}按增强、pooling 和卷积 stem 整理六组配置。

E3 到 E4 的 Grid Accuracy 中位数从 $20.56\%$ 提升至 $64.70\%$，增加 44.14 个百分点；两者 ViT 结构相同，最显著的变化来自训练分布覆盖。这是当前实验中幅度最大的改进，也说明在位置可变任务里，先让训练集包含目标扰动通常比单纯增大模型更直接。

E4 到 E5 只改变主要聚合策略，Grid Accuracy 中位数从 $64.70\%$ 提升到 $65.37\%$，增加 0.67 个百分点。五个 seed 中有四个呈现同向提升，但幅度明显小于 E3 到 E4 和 E5 到 E6，因此本报告将其作为轻微正向趋势处理。

E5 到 E6 加入卷积 stem 后，Grid Accuracy 中位数增加 15.59 个百分点，七角度平均中位数增加 16.38 个百分点。Hybrid 的提升同时出现在 Center、Large Shift、Rotation 和 Shift+Rotation，说明它并非只优化某一个特定协议。结合参数规模，291,402 参数的 Hybrid 仍小于 MLP 和 CNN，结果更可能与结构先验和训练分布匹配有关，而不只是参数数量增加。

\paragraph{结果小结}总体来看，训练分布覆盖带来最大幅度提升，Mean Pooling 的增量较小，加入卷积 stem 后 Grid Accuracy 和固定角度平均值再次明显提高。E4 同时包含平移、旋转和随机擦除，E6 则是在增强与 Mean Pooling 基础上加入 stem，因此本节只比较这些连续配置下的整体结果。

\subsection{固定角度扫描的量化结果}

随机 Rotation Accuracy 对 $[-45^\circ,45^\circ]$ 连续采样后的结果取平均，适合概括整体水平，却可能掩盖端点失效。五个 seed 下七个固定角度的宏平均分别为：MLP $39.93\%$、CNN $45.84\%$、ViT-AbsPos $41.75\%$、ViT-Aug $62.75\%$、ViT-MeanPool $64.02\%$、HybridConv-ViT $80.24\%$。

三个中心训练基线都在 $0^\circ$ 附近表现最好，并随绝对角度增大迅速下降。相比之下，ViT-Aug 和 ViT-MeanPool 的固定角度平均值提升到 $60\%$ 以上，HybridConv-ViT 的七角度平均达到 $80.24\%$，最差角度均值也保持在 $73.89\%$。增强模型不仅平均值更高，角度间波动也明显更小。正负角度并不完全对称，当前数据不足以确定这是类别方向、插值还是训练采样造成的，因此报告只把它作为可重复观察的现象，不作过度机制解释。

\subsection{类别层面的困难并未消失}

\input{tables/per_class_accuracy}

Large Shift 下，HybridConv-ViT 在 Trouser、Sandal、Sneaker、Bag 和 Ankle boot 上分别达到 $94.10\%$、$92.00\%$、$91.10\%$、$94.70\%$ 和 $94.70\%$，这些类别通常具有较明确的整体轮廓或局部特征。Shirt 只有 $49.20\%$，是 Hybrid 最困难的类别；Pullover 为 $67.90\%$，T-shirt/top 为 $72.10\%$。

增强并没有以相同幅度改善所有类别。以衬衫类为例，ViT-Aug 的准确率为 $21.70\%$，ViT-MeanPool 反而只有 $14.50\%$，说明聚合策略的总体收益不能直接推广到每个类别。相似上装之间需要依赖领口、袖长和衣身等完整形状，而边缘截断恰好会破坏这些结构。类别结果因此补充了平均指标：位置鲁棒性不仅是模型是否跟随目标移动，也取决于移动后还剩下多少可区分类别的信息。
